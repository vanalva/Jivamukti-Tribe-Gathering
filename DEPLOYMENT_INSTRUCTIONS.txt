🚀 QUICK START (Recommended)
─────────────────────────────────────────────────────────────────────────

Simply double-click: deploy-online.bat

This will:
  ✓ Start the local web server on port 8000
  ✓ Create a public ngrok URL
  ✓ Display the shareable link


📋 MANUAL DEPLOYMENT (Step by Step)
─────────────────────────────────────────────────────────────────────────

STEP 1: Start Local Server
   Double-click: start-server.bat
   
   This starts a web server at http://localhost:8000
   Keep this window open!

STEP 2: Create Public URL
   Double-click: start-ngrok.bat
   
   This creates a public URL like:
   https://your-unique-url.ngrok-free.dev
   
   Share this URL with anyone!


🛑 STOPPING THE SERVICES
─────────────────────────────────────────────────────────────────────────

Double-click: stop-all.bat

This will stop both the local server and ngrok tunnel.


📝 IMPORTANT NOTES
─────────────────────────────────────────────────────────────────────────

• The ngrok URL changes each time you restart
• Free ngrok shows a warning page (visitors can click "Visit Site")
• Both services must be running for the public URL to work
• Don't close the command windows while sharing


🔧 REQUIREMENTS
─────────────────────────────────────────────────────────────────────────

✓ Python 3.x (already installed)
✓ ngrok (already installed)


💡 TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────────

Problem: "Port 8000 already in use"
Solution: Run stop-all.bat first, then try again

Problem: "ngrok not found"
Solution: Make sure ngrok is installed and in your PATH

Problem: Website not loading
Solution: Check that both windows are still open and running


📧 SUPPORT
─────────────────────────────────────────────────────────────────────────

If you encounter issues, ensure:
1. Python is installed (check: python --version)
2. ngrok is installed (check: ngrok version)
3. Port 8000 is not being used by another application
